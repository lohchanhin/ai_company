# 像素藝術資產

## 臨時方案
目前使用彩色方塊代替像素藝術，後續會替換成真實的像素圖片。

## 資產清單

### characters/（角色）
- developer.png - 開發者角色
- database.png - 資料庫角色
- web.png - Web 伺服器角色

### objects/（物件）
- monitor.png - 螢幕
- keyboard.png - 鍵盤
- hdd.png - 硬碟

### effects/（特效）
- complete.png - 完成特效
- thinking.png - 思考泡泡
- sweat.png - 汗滴

### tiles/（地板）
- floor.png - 地板磚
- shadow.png - 陰影

## 製作工具
使用 Aseprite 或線上工具 Piskel 製作。
